# Border Radius Previewer

A tool for previewing and generating CSS border-radius values with support for both individual corners and ellipse shapes.

## Features

- Preview border radius values in real-time
- Support for individual corner radius values (top-left, top-right, bottom-right, bottom-left)
- Support for ellipse shapes (horizontal and vertical radius)
- Generate CSS border-radius code
- Copy CSS code to clipboard
- Shareable URL with current settings
